# Comparaison_Prix_2026

## Description

Comparaison de prix d'article entre deux fichiers qui peuvent être de format .xlsx ou .csv.
Donne la comparaison sous un certain format désirer, avec la possiblité d'image.

1. Installer Python

4. Exécute main.py
5. Suivez les instructions jusqu'à l'obtention du résultat
